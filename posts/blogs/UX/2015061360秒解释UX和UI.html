60秒解释UX和UI

在网页设计这个疯狂的世界里，像 UX 和 UI 这些术语时时刻刻都在聊。但这些术语精准来讲究竟是什么意思？虽然 UX （用户体验）和 UI （用户界面）设计都是呈现给用户最棒的上网体验的重要组成成分，但是它们其实是不一样的。现在我们来快速分析一下两者最主要的不同和聊一下两者的整合。

UI 主要包含了网页视觉设计的各方面。通过开发不同层的展示和互动，使产品有一个整体外观和感觉。 Usability.gov 是一个很好的资源，里面有很好的实践、指导、以及不同的 UI 元素，例如：

  • 输入：输入框、日期选择、下拉列表、复选框、列表框等

  • 导航：滑动条、搜索栏、分页、侧边栏、标签、图标等

  • 分享：朋友列表、“成为粉丝”按钮、赞、分享按钮、邀请朋友等

  • 信息：文本内容、工具提醒、信息框、通知、图标、进度栏等

UI 设计不仅包括你所看到的，也包括了用户与网站每个元素互动的设计。例如下面的天气 APP 是 UI 设计的互动成分的典型例子。

如果 UI 包括了以上元素，那么 UX 相比 UI 还多了什么？用户体验设计被定义为通过改善可用性、易用性和提供产品与用户间互动时的乐趣来提高用户的满意度和忠诚度的方法。为了创建公司与用户间高质量的体验， UX 设计师专注于研究、可用性测试以及了解用户其他方面的方法。举个例子，许多 UX 研究人员利用 A/B 测试来找出两个设计中哪个设计与用户有更好的共鸣。

那么，这两者间究竟有什么关系？令人深刻的 UI 和 UX 设计彼此之间概念是可互换的，都是为用户创建一个整体的愉悦体验。如果它们其中之一没达到标准，这会严重影响网站的效果。举个例子，如果一个网站有很赞的 UI 但有很差劲的 UX ，这样这个网站其实是体验不好的，因为用户会觉得这个网站看起来很棒但用起来感觉很费劲。另一方面，如果一个产品非常好用但看起来很糟糕就是很好的 UX 和很差的 UI.

附：虽然 TVB 翻译完后还是觉得作者没表达清楚 UI 和 UX 的差别在哪，在此 TVB 用 5s 来解释一下什么是 UX 、 UI 和 IxD （交互设计）。

UI 就是一个人的颜值（你喜欢哪种？）

IxD 就是这个人品质和行为(听说大家都喜欢大白这种类型的暖男，前提是不长成这样的)

UX 就是 UI 和 IxD 还有其他因素结合后这个人适合不适合交往了
